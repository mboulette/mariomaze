MARIO MAZE

Bienvenu dans mon petit jeu imitation de Sper Mario Bros.
Le but est de libérer la princesse de Bowser dans le moins de temps possible.

Contrairement à mario, vos boules de feu sont comptés, et vous devez ammasser des clés pour ouvrir les portes.

Voici les touches pour jouer:

[←][→] Pour vous déplacer
[Z] Pour courrir
[X], [↑] [SPACEBAR] Pour sauter
[C] Pour lancer des boules de feu
[P] Pour mettre le jeux sur pause
[↓] Pour vous pencher
[↑] ou [↓] Pour ouvrir des portes, entrer dans les tuyeaux et grimper aux échelles

Appuyer sur [ENTER] pour commencer.
